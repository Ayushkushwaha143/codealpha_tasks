# Iris Flower Classification 🌸📊

This project is part of my **Data Science Internship at CodeAlpha**.
The objective of this task is to build a **Machine Learning model that classifies iris flowers** into different species based on their features.

---

## 📌 Task

**Task: Iris Flower Classification**

The goal of this project is to predict the **species of an iris flower** using features such as:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

The model classifies flowers into three species:

* Setosa
* Versicolor
* Virginica
