# 📌 Task: Iris Flower Classification 🌸

## Objective

The objective of this project is to build a **machine learning classification model** that predicts the species of an iris flower based on its physical features.

## Dataset Features

The dataset contains the following features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

## Target Classes

The model classifies flowers into three species:

* Setosa
* Versicolor
* Virginica

## Steps Performed

1. Imported required Python libraries
2. Loaded the Iris dataset
3. Performed data preprocessing
4. Conducted Exploratory Data Analysis (EDA)
5. Split dataset into training and testing data
6. Trained a machine learning classification model
7. Evaluated the model performance
8. Predicted iris flower species

## Result

The model successfully classifies iris flowers into their correct species using machine learning techniques.

# 📌 Task: Car Price Prediction 🚗

## Objective

The objective of this project is to build a **machine learning regression model** that predicts the selling price of a car based on different features.

## Dataset Features

Some important features used in the dataset include:

* Car Name
* Year
* Present Price
* Driven Kilometers
* Fuel Type
* Selling Type
* Transmission
* Owner

## Steps Performed

1. Imported required Python libraries
2. Loaded the car dataset
3. Cleaned and preprocessed the data
4. Performed Exploratory Data Analysis (EDA)
5. Selected relevant features
6. Split dataset into training and testing data
7. Trained a regression model
8. Evaluated model performance
9. Predicted car prices

## Result

The machine learning model successfully predicts the selling price of cars based on input features.
