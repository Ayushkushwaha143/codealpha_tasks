# Car Price Prediction 🚗📊

This project is part of my **Data Science Internship at CodeAlpha**.
The objective of this task is to build a **Machine Learning model that predicts car prices** based on various features such as year, fuel type, transmission, and other attributes.

---

## 📌 Task

**Task: Car Price Prediction using Machine Learning**

The goal of this project is to analyze car data and develop a model that can predict the **selling price of a car** using different input features.

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib / Seaborn
* Scikit-learn
* Jupyter Notebook

---

## 📊 Project Workflow

1. Import required libraries
2. Load the dataset
3. Data cleaning and preprocessing
4. Exploratory Data Analysis (EDA)
5. Feature selection
6. Train-test split
7. Model training (Regression Model)
8. Model evaluation
9. Car price prediction
